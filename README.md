# 工作日报润色助手 - README

一个现代化的Web应用,帮助您将简短的工作笔记转换为专业的Markdown格式日报,使用智谱AI的GLM模型进行内容润色。

## 快速开始

1. **配置API密钥**
   - 访问 [智谱AI开放平台](https://open.bigmodel.cn/) 获取API Key
   - 打开应用,点击右上角"设置"按钮
   - 输入API Key并保存

2. **使用应用**
   - 在左侧输入框中输入简短的工作内容(每行一项)
   - 点击中间的转换按钮
   - 在右侧查看润色后的Markdown格式内容
   - 点击复制按钮一键复制

## 功能特点

- ✨ **AI智能润色**: 将简短笔记扩展为专业完整的句子
- 🔄 **独立转换**: 今日完成和明日计划各自独立转换,互不影响
- 📈 **累加模式**: 转换内容自动追加,支持逐步完善日报
- 📝 **Markdown输出**: 自动生成标准Markdown列表格式,带项目符号
- ✏️ **可编辑输出**: 点击右侧输出区域可直接编辑内容
- 💾 **手动保存**: 编辑后可点击保存按钮将修改保存到本地
- 📋 **一键复制**: 快速复制转换后的内容
- 🗑️ **清空功能**: 可随时清空输出区域重新开始
- 🎨 **现代化UI**: 深色主题,玻璃态效果,流畅动画
- 🔒 **本地存储**: API密钥和日报内容安全存储在浏览器本地
- 🚀 **最新模型**: 支持GLM-4-Flash/Air/AirX/Long/Plus等最新模型

## 技术栈

- 纯HTML/CSS/JavaScript,无框架依赖
- 智谱AI GLM模型集成
- 响应式设计,支持多种设备

## 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `app.js` - 核心逻辑

## 使用示例

**输入**:

```
修复登录bug
优化首页性能
```

**输出**:

```markdown
- 完成了用户登录功能的bug修复,解决了验证码失效的问题
- 对首页加载性能进行了优化,将首屏渲染时间缩短了30%
```

## 本地运行

### Windows 用户

双击运行 `启动.bat` 文件即可自动启动本地服务器，然后访问：

- <http://localhost:8080>
- <http://127.0.0.1:8080>

### 手动启动

如果没有 Node.js 环境，也可以直接在浏览器中打开 `index.html` 文件使用。

### 使用飞书同步功能

如需使用飞书同步功能，请运行 `启动日报助手.bat`，它会同时启动代理服务和网页服务。
